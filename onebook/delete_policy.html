계정 및 데이터 삭제 안내

저희는 귀하의 개인 정보를 소중히 여기며, 책임감 있게 처리하고 있습니다.

"원서한권" 앱의 회원정보에서 직접 탈퇴 및 데이터 삭제가 가능합니다.

 

만약 앱 내에서 직접 처리하기 어려운 경우, 다음과 같은 방법으로 삭제 요청을 하실 수 있습니다:

로그인 시 사용한 이메일 주소로 exceed@exceedventures.co.kr로 삭제 요청을 보내주시기 바랍니다.

요청을 접수한 후, 3영업일 이내에 데이터 삭제를 진행하겠습니다.

저장된 회원 정보는 이메일 주소와 이름이며, 삭제 시 별도의 저장 기간 없이 즉시 삭제 처리됩니다.

사용자가 작성한 댓글 컨텐츠도 삭제 시 별도의 저장 기간 없이 즉시 삭제 처리됩니다.

기타 문의 사항이나 도움이 필요하신 경우, exceed@exceedventures.co.kr로 연락해 주세요.
