제목: DNA 데이터 저장장치가 삼성전자와 SK하이닉스를 위협할까?

항상 최신 기술 개발 상황은 모니터링할 필요가 있다고 생각합니다. 
따끈한 신상 기술 하나를 정리해 봅니다. 
컴퓨터의 언어는 2진법임. 
모든 데이터(사진 영상 텍스트)를 0과 1이라는 두 개의 숫자로 변환시켜 하드디스크에 저장함. 
컴퓨터가 데이터를 처리하는 방식과 생명체가 유전 정보를 저장하는 방식은 놀랍도록 비슷함. 
생명체의 언어는 4진법임. 
DNA는 아데닌(A) 시토신(C) 구아닌(G) 티민(T)이라는 네 가지 염기로 구성이 됨.  
컴퓨터의 0과 1을 DNA의 네가지 염기에 대응시킬 수 있음. 
00은 A 01은 C 10은 G 11은 T로 매칭하는 식임. 
C-A-T-G라는 코드가 완성되면 데이터를 인공 DNA 가닥으로 만들 수 있음. 
인공 DNA는 빛과 수분이 차단된 캡슐에 보관하면 됨. 
저장한 데이터가 필요하면 DNA 해독기로 DNA 가닥의 순서를 읽어내고 이것을 0과 1로 변환시킴.
0과 1로 변환된 데이터는 현재의 컴퓨터 데이터처럼 모니터에 띄울 수가 있음. 
DNA로 만든 데이터 스토리지(하드디스크 SSD 클라우드등 데이터 저장장치)가 만들어지는 것임.
DNA 데이터 저장장치는 기존 저장 장치(SSD HDD등)에 비해서 압도적인 장점이 몇 가지 있음. 
첫 번째는 미친듯한 저장능력임. 
이론적으로 DNA 1g에는 215 페타 바이트(PB)의 데이터를 저장할 수 있음. 
1PB(페타) = 1000TB = 1000000GB임. 
122일의 기적이라고 부르는 일론 머스크의  xAI 콜로서스 데이터센터가 있음. 
테네시주 멤피스에 만들어진 xAI의 데이터센터는 구글 메타 MS등이 몇 년에 걸쳐서 할 일을 몇 달 만에 해치운 사례임. 
데이터센터를 지으려면 설계 전력망 확보 건물 건축 장비 반입까지 빨라도 3~4년은 걸리는 것이 상식임. 
일론 머스크는 건물을 짓는 대신 테네시주 멤피스에 있는 거대한 가전제품 제조공장을 사들여 데이터센터를 만들기 시작함. 
2024년 봄에 프로젝트를 발표하고 122일 만인 2024년 8월 8일에 10만 개의 H100 GPU 클러스트를 최초로 가동함. 
일론 머스크는 10만 개의 H100 GPU를 가동한 이후 92일 만에 이것을 20만 개로 2배 증설을 함. 
인근 부지와 건물을 추가 매입해서 콜로서스2 콜로서스3등으로 확장하는 중이고 100만개의 GPU를 넣는 것이 목표임.
정확한 비용은 xAI의 내부 기밀이지만 업계가 추산하는 비용은 다음과 같음. 
콜로서스 1 2 3 세개의 건물과 100만 개의 GPU 2GW 규모의 발전시설등 총 투자액은 350억 달러 이상이 들어갔을 것이라고 함. 
지역 전력망이 데이터센터가 사용하는 전력을 감당 못하자 데이터센터 옆에 발전소를 설치해 전기까지 자체 조달하기 시작함. 
콜로세스 1의 스토리지 총용량이 1000PB이고 2와 3을 모두 합치면 5000PB급으로 추정하고 있음. 
DNA는 5000PB의 데이터를 밥 한 숟가락 분량인 20g에 저장할 수 있음. 
캡슐에 담긴 DNA에 전기코드를 꽂을 필요가 없으니 유지 전력은 0와트임. 
수명도 차이가 있음. 
HDD나 SSD는 5~10년이면 부품이 마모되거나 자성이 약해져서 데이터를 잃을 위험이 있음. 
GPU는 수명이 더 짧아서 3년이면 27% 정도가 고장 나는 것으로 Meta가 자체 분석함. 
매머드 화석의 DNA를 지금도 추출할 수 있듯이 DNA는 수백~수천 년간 데이터가 보존됨. 
DNA 데이터 저장장치의 선두주자는 마이크로소프트임. 
마이크로소프트(MS)는 미국 워싱턴대학과 협력해서 200MB의 데이터를 DNA에 저장하고 오류 없이 100% 복원하는데 성공함. 
카탈로그라는 바이오 스타트업은 영어 위키피디아 전체 텍스트를 시험관 속 DNA 잉크 한 방울에 저장하는 프로젝트를 성공시킴.
여기까지만 들어보면 삼성전자와 SK하이닉스가 큰일이 난 게 아닌가 생각할 수 있음. 
속도에 문제가 있어서 당장은 위협이 되지 못할 것 같음.  
DNA는 데이터를 쓰고 읽는 속도가 느림. 
현재 기술로는 데이터를 쓰고 읽는데 일 단위 시간이 걸린다고 함.  
데이터를 실시간으로 사용하는 용도로는 DNA 스토리지를 쓸 수 없는 것임.
자주 꺼내보는 핫 데이터(Hot Data)가 아니라 안전하고 오래 보관해야 하는 콜드 데이터(Cold Data) 시장은 대체 가능해 보임. 
읽고 쓰는 속도가 너무 느린 부분을 해결하는 대책으로는 DNAformer등의 신기술이 별도로 연구되고 있음. 
DNAformer는 이스라엘 테크니온 공대 연구팀이 개발해서 2025년 3월 네이처에 발표한 기술임. 
DNAformer는 DNA를 읽을때 AI의 딥러닝을 활용해서 속도와 정확성을 향상시키는 기술임. 
기존 방식대비 3200배 빠른 속도로 100MB 데이터를10분만에 읽는데 성공했고 AI성능이 올라가면 더 빠른 속도도 가능해 짐. 
속도뿐만 아니라 읽는 과정에서 발생하는 염기삭제 치환 순서등의 에러도 잡아낼 수 있어서 정확성도 향상됨.
여러 신기술들이 동시에 연구되고 있어서 관심은 가지고 지켜볼 가치는 있는 단계임. 
2025년 5월 Twist Bioscience가 DNA 데이터 저장 사업부를 독립법인인  Atlas Data Storage로 분사한 사례도 있음. 
제프 베이죠스 미국 정보기관 CIA의 벤처펀드인 In-Q-Tel등이 투자를 함. 
Atlas Data Storage는 2025년 12월 3일 AMIA(영상협회)컨퍼런스에 참석해서 공개 시연을 함. 
캡슐 뿐만 아니라 저장장치등을 모두 포함된 1리터 크기 용기에 60PB(영화 66만편)를 수천년간 저장할 수 있다는 것임.  
AI 학습 데이터셋 장기보존이 필요한 하이퍼스케일러 영상 보관기관 데이터 장기보관이 필요한 금융 의료기관등이 타깃이라고 함. 

한 줄 코멘트. 
하버드 와이스 연구소는 "전 세계 디지털 데이터 전체를 81kg의 DNA에 저장할 수 있다"라고 밝히고 있음. 전기도 필요없고 수천 년간 안정적으로 보관할 수 있는 DNA 데이터 스토리지도 상용화되면 나름의 시장을 확보할 수 있을것 같음. 실시간으로 꺼내볼 필요는 없지만 오랜 기간 안정적으로 보관이 필요한 데이터 수요도 있기 때문임. 그런데 갑자기 돌솥밥이 생각남. 여의도 너섬 청국장에 따라나오는 돌솥밥이나 먹을까..
